## What Generate Lorem does

**Generate Lorem** fills eligible selected text layers with lorem ipsum text.

It is designed for fixed-width or auto-height text layers where placeholder copy needs to fit the current text style and box.

## Relaunch support

Eligible text layers can expose a relaunch shortcut for lorem generation.

The plugin also tracks generated lorem nodes and can refresh generated copy when relevant text-layer changes happen.

## When to use it

Use Generate Lorem when testing:

- text style rhythm
- card content density
- responsive text boxes
- layout width decisions
- line-height and spacing relationships

Use real content before final handoff.
