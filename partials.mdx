---
title: "Partials"
---

Partials are reusable views. To reference a `/resources/views/_card.antlers.html` You don't have to enter the underscore `_`

<CodeGroup>

```jsx home.antlers.html
{{ partial:card }}
```

```html _card.antlers.html
<p>This is a beautiful card!</p>
```

</CodeGroup>

When it's in a subdirectory like:

`/resources/views/parts/_card.antlers.html`

```javascript
{{ partial:parts/card }}
```

## Passing Data

```javascript
{{ partial:card favoriteFood="Cookies" }}
```

```jsx _card.antlers.html
<p>My favorite food is: {{ favoriteFood }}</p>
```
