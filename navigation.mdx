Create a new Navigation, and give it a name like `Header`. Look carefully at the handle name. (This is standard the same as your name, but lowercase).

## Configuring your navigation

Choose the Collections you'd want to be selected. Like Pages.

## Adding Links

If you have a Collection selected, I'd recommend `Link to Entry` and selected the Items you want in your navigation.

## Showing the links in Antlers

```jsx
{{ nav:links }}
<a src="{{ url }}">{{ title }}</a>
{{ /nav:links }}
```

## Check if the current page is the link

Sometimes you want to some styling if it's the current page. Like showing that font in bold.

```jsx
{{ nav:links }}
    <a src="{{ url }}" {{ if is_current }} class="current" {{ /if }}>
        {{ title }}
    </a>
{{ /nav:links }}
```
